# IsHashValid

A simple hash validator that allows you to do complex validations with custom error messages.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'is_hash_valid'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install is_hash_valid
